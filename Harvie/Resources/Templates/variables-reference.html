Template Variables Reference

All variables, sections, conditionals, and filters available in Harvie templates.

This file is auto-generated and will be overwritten on app updates. It is placed in your Templates folder for easy reference while editing templates.

Syntax

Variables

Use double curly braces to output a value:

{{invoice.number}}
{{client.name}}

Sections (Loops)

Iterate over arrays with {{#section}}...{{/section}}:

{{#lineItems}}
  <tr>
    <td>{{description}}</td>
    <td>{{amount}}</td>
  </tr>
{{/lineItems}}

Inside a section, variables resolve against the current item.

Conditionals

Show or hide blocks based on a boolean value:

{{#if invoice.hasDiscount}}
  <tr><td>Discount</td><td>{{invoice.discountAmount | currency}}</td></tr>
{{/if}}

Filters

Apply formatting to a value with the pipe | syntax:

{{invoice.amount | currency}}
{{invoice.issueDate | date:"dd.MM.yyyy"}}
{{invoice.tax | number:1}}
{{invoice.notes | markdown}}

Filters Reference

FilterDescriptionExample
currency Formats a number with grouping separators and two decimal places {{invoice.amount | currency}} → 9'427.08
date:"format" Formats a date using the given format string {{invoice.issueDate | date:"dd.MM.yyyy"}} → 01.03.2026
number:digits Formats a decimal with the specified number of decimal places {{invoice.tax | number:1}} → 8.1
markdown Converts markdown to HTML (**bold**, *emphasis*, - lists, line breaks) {{invoice.notes | markdown}}
